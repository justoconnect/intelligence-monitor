# Intelligence Monitoring Agent 🇬🇦

An intelligent web monitoring system that tracks developments across four key sectors in Gabon:
- **Transport (Gabon)** - Infrastructure, railways, ports, airports
- **Transport (Africa)** - Regional corridors and connectivity initiatives
- **PNCD** - National Community Development Program
- **Mining (Gabon)** - Sector developments and project updates

## Features ✨

- ✅ **Web Source Monitoring** - Automated intelligence gathering from multiple sources
- ✅ **Deduplication** - SHA256-based content hash to prevent duplicates
- ✅ **SQLite Database** - Efficient storage with indexing for fast queries
- ✅ **Smart Search** - Keyword-based search across titles, summaries, and entities
- ✅ **JSON Export** - Export all intelligence data in structured format
- ✅ **Category Analysis** - Break down entries by sector
- ✅ **Impact Scoring** - 1-10 impact scale for prioritization
- ✅ **Full Audit Trail** - Timestamp tracking and source attribution
